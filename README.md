# 使用方法：
1、每一章（也就是每一个\Chapter）后，需要在“\markboth{第一章\quad 章标题}{}”中更改自己的章标题；
2、在中文摘要中：“\markboth{论文标题}{}”填写自己的论文标题，英文摘要不用管；
